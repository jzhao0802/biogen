Set up your data folder to create experiments

1. First, mount your data folder in Windows command prompt:
net use D: \\localhost\C$\Users\npoh\Documents\myProjects\Biogen_Tecfidera

If you run this on the server, use
net use D: \\kgxsapp100\F\Projects\Biogen_Tecfidera

In this folder, make sure that 'Data' and 'Results' folder are accessible.

2. Setup the working direcotry (optional)

net use D: \\localhost\C$\Users\npoh\Documents\Git\projects\Biogen_Tecf2017_round5
net use W: F:\Norman\Biogen_Tecf2017_round5\loading
